# Overworld - Alien Ecology Simulation

A tile-based browser game  where you fight to survive on a realistically simulated alien world. The environments, flora and fauna of this planet are modeled using physics, plausability and especially evolutionary principles- especially in the context of adaptive radiation. This game is heavily inspired by Biblaridion's incredible _Alien Biospheres_ series, a fascinating and rich speculative evolution project. Other strong influences come from classic roguelikes such as _Nethack_ and _Caves of Qud._

A big fat work in progress, by the way. Check the Todo document for rough progress.
